帮助
快速开始
1
按住热键
默认 RightCtrl，按住不放。浮动窗口出现即表示开始录音。
2
对着麦克风说话
正常音量即可。开启实时转写后，文字会实时显示在浮动窗口中。
3
松开热键
松开即触发完整转写（ Whisper ），如有开启 LLM 纠错会随后自动处理。
4
文字自动粘贴
转写结果通过剪贴板 + Ctrl+V 注入当前焦点应用。如开启「粘贴前确认」，会先弹出编辑窗口。
使用技巧与常见问题
切换识别语言
在「通用」设置页选择语言，更改后立即生效，无需重启。
实时转写不显示文字
确认「通用」设置中已开启「实时语音转写」开关；同时确保模型已下载完成。
模型下载速度慢
在「模型」设置页切换下载镜像。国内网络推荐 HF-Mirror，国际网络可用 HuggingFace。
LLM 纠错未生效
检查「LLM」设置页中的 API 地址、密钥、模型名称是否填写正确；点击「测试连接」验证连通性。
关闭开机自动启动
在「通用」设置页关闭「开机自动启动」开关，保存后生效。
热键与其他软件冲突
在「快捷键」设置页更换为其他按键。注意：Windows 的 Fn 键在固件层处理，无法被本软件识别。
如何保存训练数据
在「数据」设置页开启「保存训练数据」，选择保存目录。每次录音后会自动保存 WAV 音频和 JSON 元数据。
已知问题与限制
管理员权限应用无法注入文字
如果目标窗口以管理员权限运行（如任务管理器、某些 IDE），本软件无法向其粘贴文字。这是 Windows 安全模型的限制。
剪贴板被占用时可能延迟
某些应用（如截图工具、Office 套件）可能长时间占用剪贴板。本软件已设置 2 秒超时保护，超时会自动报错恢复。
实时转写准确性较低
实时转写基于 5 秒滑动窗口的短音频片段，准确性略低于热键松开后对完整录音的全量识别。最终文本以全量识别为准。
GPU 模式首次加载耗时
首次启用 GPU 加速时，Vulkan 驱动需要编译着色器缓存，耗时约 10-30 秒。后续启动会快很多。
多显示器下浮动窗口位置异常
跨显示器使用时，浮动窗口偶尔出现在非预期的显示器上。点击系统托盘菜单中的「重置状态」可立即恢复。
